# üõí Deals N' Discounts
Deals N' Discounts is a full stack e-commerce application. Users can browse products, search different categories, add items to a cart, and order items in their cart. Once ordered, the items stock is updated in the database. This project was developed for Millersville University's CMSC 421(Advanced Web Development). 

# ‚öôÔ∏è Installation
Paste "https://codesandbox.io/p/sandbox/serene-tamas-d6gnqp" in a browser.

If running locally run

```bash
    git clone https://github.com/CollinDonnan/Deals-and-Discounts.git
    npm install yarn --global
    yarn install
    node app.js
```

For database features to work a .env file with a uri variable containing the uri string must be added
